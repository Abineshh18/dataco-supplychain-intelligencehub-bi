# Data Source: DataCo Smart Supply Chain for big data analytics

This project utilizes the "DataCo Smart Supply Chain for big data analytics" dataset, which consists of transaction data from a supply chain company. 

Dataset Links

Primary Source: [Kaggle - DataCo Smart Supply Chain](https://www.kaggle.com/datasets/shashwatwork/dataco-smart-supply-chain-for-big-data-analysis)
Format: CSV (Comma Separated Values)

Dataset Overview

The dataset contains roughly "180,000+ rows" and "52 columns" of data spanning 2015 to 2018. It covers various aspects of the supply chain process, including:
Provisioning: Product details and warehouse sourcing.
Production: Manufacturing departments and categories.
Sales: Customer demographics, order pricing, and profit margins.
Commercialization: Delivery status, shipping modes, and geographic regions.

Usage in this Project :

For this portfolio project, the raw data was cleaned and transformed into a "Star Schema" using SQL and Power Query to optimize for Business Intelligence reporting.
